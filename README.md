1. Student Grade Calculator

## Project Description
This Python program calculates student grades based on marks entered by the user.
It validates input, assigns grades, and shows encouraging messages.

2. Features
- Input validation using while loop
- Grade calculation using if-elif-else
- Modular code using functions
- Beginner-friendly and readable

3. Grading Logic
A: 90–100  
B: 80–89  
C: 70–79  
D: 60–69  
F: 0–59  

4. How to Run
1. Open terminal
2. Run: python grade_calculator.py
3. Enter student name and marks
